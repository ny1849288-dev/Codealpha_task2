# 📧 Email Extractor using Python (CodeAlpha Internship)

This project is a Python-based automation script that extracts all email addresses from a text file and saves them into a separate file.

## 🎯 Objective
To automate the process of finding and collecting email addresses from a file using Python.

## 🚀 Features
- Extracts all email addresses using Regular Expressions (Regex)
- Removes duplicate emails
- Handles file errors gracefully
- Saves extracted emails into a new file

## 🛠️ Technologies Used
- Python
- re (Regular Expressions)
- os (File handling)

## 📂 How It Works
1. User provides the input file name (e.g., `data.txt`)
2. The program reads the file content
3. Extracts all email addresses using regex
4. Removes duplicates
5. Saves results in `extracted_emails.txt`
